1º ponto

Alojamento de página Web na cloud

Aprendemos a :

  • Criar ficheiros HTML
  • Alguns comandos da linguagem de marcação
  • Criar repositório no git
  • Alocar o nosso site em um servidor
  • Por fim, criámos o nosso primeiro web site com ajuda do professor

2º ponto

Conhecer a Internet

Aprendemos a :

  • Obter o ip dos nossos equipamentos e servidores


  • Verificar e pereber o percurso efectuado pelo nosso pedido ao servidor


3º ponto

Acesso via HTTP à minha página Web

Aprendemos a :

  • Ver o Código fonte da nossa ou qualquer outra pagina web, mensagens HTTP GET e HTTP POST

  • Inspecionar o código, verificar o disparar de ações ao clicar em um site, verificar os timmings de resposta e de espera e conexão Verificar a totalidade de ficheiros disparados em um unico acesso, verificar o tamanho dos recursos Verificar o request HTML e o método request, se é GET ou POST. Aprendemos que o Inspecionar de uma pagina web nos consegue trazer toda a informação necessária para perceberemos de onde vem, o que trás com o que lida, os seus arquivos e código javascript possivel de ver em preview.
